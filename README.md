## Description
Repo นี้เก็บโปรแกรมตัวอย่างภาษา C++ เพื่อช่วยในการเรียนรู้หลักการพื้นฐานอย่าง:
- การรับ-ส่งข้อมูล  
- เงื่อนไข (if, switch)  
- การทำงานกับลูป (for, while)  
- ฟังก์ชัน และ Array

## Features
- ตัวอย่างโค้ดง่ายต่อการดูและแก้ไข  
- เหมาะสำหรับผู้เริ่มต้นเรียนรู้พื้นฐานภาษา C++
